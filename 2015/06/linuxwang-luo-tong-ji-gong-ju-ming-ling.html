Linux网络统计工具/命令

Published: Mon 08 June 2015
By linux fans from china

In 系统运维.

Author: nixCraft

我在Linux(基于CentOS 或者 Debian 的发行版)中该如何查看当前网络端口吞吐量的统计信息？在Linux操作系统中如何查看当前内核snmp计数器以及网络端口的统计信息？

你可以使用以下任何一个命令：

# nstat

ss 命令 : 查询套接字工具

要查看统计汇总, 输入:

# ss -s

输出样例:

查看ss 命令教程 获取更多信息。

netstat 命令 : 不错的显示套接字信息的旧工具

查看所有网络端口以及汇总信息表，输入：

# netstat -i

ip / ifconfig 命令 : 配置或者显示网络端口信息

键入以下命令:

# ifconfig

或者

# /sbin/ifconfig

或者

# ifconfig eth0
